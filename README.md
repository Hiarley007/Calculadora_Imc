# 📊 Calculadora de Índice de Massa Corporal (IMC)

Este projeto é um programa simples desenvolvido em **Python** que calcula o **IMC (Índice de Massa Corporal)** a partir dos dados informados pelo usuário e classifica o resultado conforme os parâmetros da Organização Mundial da Saúde (OMS).

---

## 🚀 Funcionalidades

- Solicita o **nome** do usuário  
- Solicita o **peso (em Kg)**  
- Solicita a **altura (em metros)**  
- Calcula automaticamente o IMC  
- Exibe a classificação correspondente ao resultado  

As categorias apresentadas pelo programa são:
- Abaixo do peso  
- Peso normal  
- Sobrepeso  
- Obesidade Grau I  
- Obesidade Grau II  
- Obesidade Grau III  

---

## 🧮 Fórmula Utilizada

O cálculo do IMC é feito utilizando a seguinte fórmula:

IMC = peso / (altura * altura)


## 📌 Classificação Utilizada

| IMC            | Classificação         |
|----------------|----------------------|
| ≤ 18.5         | Abaixo do peso       |
| 18.6 – 24.9    | Peso normal          |
| 25 – 29.9      | Sobrepeso            |
| 30 – 34.9      | Obesidade Grau I     |
| 35 – 39.9      | Obesidade Grau II    |
| ≥ 40           | Obesidade Grau III   |

---
